> **overflow**
  내부의 요소가 부모의 범위를 벗어날 때 어떻게 처리할 지 지정

**overflow: hidden;**
  영역을 벗어나는 부분을 안 보이게 함
**overflow: scroll;**
  영역을 벗어나는 부분을 스크롤로 만듦
>

```css
p {
      width: 200px;
      height: 150px;
      border: 3px solid black;
      margin-bottom: 100px;
    }
```
